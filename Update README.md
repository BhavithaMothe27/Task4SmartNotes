# Task4SmartNotes

Overview:

Task4SmartNotes is a simple and modern Android Notes application developed using Kotlin and Jetpack Compose. The application allows users to create, view, and delete notes with an intuitive and responsive user interface.

Features:

* Add Notes
* View Notes List
* Delete Notes
* Dark Mode Toggle
* Material 3 UI Design
* Responsive Layout

Technologies Used:

* Kotlin
* Jetpack Compose
* Android Studio
* Material 3

Learning Outcomes:

* State Management in Jetpack Compose
* Modern Android UI Development
* Building Responsive User Interfaces
* Android App Architecture Basics
